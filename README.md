🏥 MedFlow AI — Smart Pharmacy Inventory & Expiry Management

Smarter inventory. Zero waste. Always in stock.

MedFlow AI is an AI-powered healthcare operations platform designed to optimize pharmacy inventory management safely and efficiently. The system focuses on reducing medicine wastage, preventing stock shortages, and automating manual workflows using AI and Machine Learning — without involving patient diagnosis or treatment decisions.

🚀 Features

📄 AI Invoice Parser

Upload supplier PDF invoices and automatically extract:

Medicine Name

Batch Number

Quantity

Expiry Date

The extracted data is automatically added to the inventory system, reducing manual work and human errors.

⏳ Smart Expiry Tracking

Track medicines expiring within:

30 Days

60 Days

90 Days


Get real-time alerts and smart redistribution suggestions to minimize medicine wastage.



📈 ML Demand Forecasting



Machine Learning models analyze historical inventory and sales data to:



Predict future medicine demand

Prevent stockouts

Recommend optimal restocking quantities

📊 Professional Admin Dashboard

Modern SaaS-style dashboard with:

Inventory analytics

Expiry alerts

Demand prediction charts

Waste reduction insights

Redistribution recommendations
